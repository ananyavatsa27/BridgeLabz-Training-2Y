Problem 2: Word Search (DREAM)

a) Backtracking Pseudocode:
function search(board, word, i, j, index):
    if index == len(word):
        return true
    if out of bounds or mismatch:
        return false

    temp = board[i][j]
    board[i][j] = '#'  # mark visited

    for each direction:
        if search(next cell):
            return true

    board[i][j] = temp  # backtrack
    return false
